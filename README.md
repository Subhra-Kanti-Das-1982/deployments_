# DhaaRAa - Dharma Reflection Assistant 
#### DhaaRAa is a personal assistant based on Indian Hindu Mythology and scriptures of wisdom to assist adolescents and teenagers to self analyse their own actions and behavior in a systematic approach without losing their self-esteem or confidence and still be guided on the principles of righteousness.
#### Please find the slide deck in the repository which gives a comprehensive idea to use the tool and the principle of working of the framework.
#### The source codes are completely open and license free. Anyone is free to download, modify the code base and make commit back to the repository.
#### The only request to interested users is to cite this work wherever they are making use of this or deploying this framework. The idea is to help as many teenagers and adolescents across the globe as possible.
#### This will remain a very active repository and we will constantly update the codebase with further recommendations and advanced features.
